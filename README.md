# 🃏 How To Play

## Goal

Build the highest value hand in a single suit.

Maximum score:

**31**

Example:

- Ace = 11
- King = 10
- Queen = 10

Total = 31

## Card Values

| Card | Value |
|--------|--------|
| Ace | 11 |
| King | 10 |
| Queen | 10 |
| Jack | 10 |
| Number Cards | Face Value |

## On Your Turn

Players may:

- Draw from deck
- Draw from discard pile
- Discard a card
- Knock
- Stand

## Round End

- Lowest hand loses a coin
- Failed knocker loses two coins

## Game End

Last player remaining wins.

# 💻 Install As An App (Windows PC)

## Google Chrome

1. Open:
   https://trethacker.github.io/31/

2. Wait for the page to fully load.

3. Click the Install icon in the Chrome address bar.

4. Click **Install**.

5. Thirty-One will be installed like a desktop application.

6. Launch from:
   - Desktop shortcut
   - Start Menu
   - Taskbar (if pinned)

---

# 🍎 Install As An App (iPad / iPadOS)

1. Open Safari.

2. Navigate to:

   https://trethacker.github.io/31/

3. Tap the Share button.

4. Select:

   **Add to Home Screen**

5. Tap **Add**.

6. The game will appear on the Home Screen and launch like a native app.
